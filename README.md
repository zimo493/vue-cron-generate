# vue-cron-generate
基于Vue3、TypeScript、NaiveUI的Cron表达式生成器

## 依赖

- Vue 3.0.0+
- Naive UI / Element Plus
- ...

## 安装使用步骤

- **Clone：**

```shell
# Gitee
git clone https://gitee.com/zimo493/vue-cron-generate.git

# GitHub
git clone https://github.com/zimo493/vue-cron-generate.git
```

- **Install：**
```shell
pnpm install
```

- **Run：**
```shell
pnpm dev
```
